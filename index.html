Worst-Case Preset Note

						
Worst-case preset not selected yet. Choose the preset and build steps to see why sorting networks still apply every comparator in the same fixed order.

					


					

						Overview
						
A sorting network is a fixed wiring diagram of compare-swap units. Bitonic networks are data-oblivious: the comparator structure does not change with input values, only swap decisions do.

					


					

						Worked Example
						
Try the 0-1 preset or worst-case preset, then build steps. Track how stages form bitonic subsequences before final bitonic merges produce a globally sorted output.

					


					

						
Key Concepts

						
    
							
  • Bitonic sequence: first monotone up then down (or reverse).
    • 
							
  • 0-1 Principle: passing all binary inputs implies correctness for all inputs.
    • 
							
  • Complexity: O(n log^2 n) comparators and O(log^2 n) parallel depth.
    • 
							
  • Data-oblivious structure enables parallel hardware scheduling.
    •
